# Stop AI Repetitive Code Scans

This example demonstrates the inefficiency of AI agents repeatedly scanning codebases as plain text (like `grep`). It contrasts a 'Naive AI Agent' that performs full file scans for every query with a 'CodeGraph AI Agent' that builds an initial index and then answers subsequent queries instantly without re-scanning, showcasing how to optimize resource usage.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`
